# Contributing to DashScene

Thank you for your interest in contributing to DashScene.

Please refer to the [driftsys organization contributing guidelines](https://github.com/driftsys/.github/blob/main/CONTRIBUTING.md) for general contribution policies, commit conventions, and coding standards.

## Repository-Specific Notes

- This is a Cargo workspace with multiple crates under `crates/`.
- Use `just check` to run the full verification suite before submitting a PR.
- Follow [Conventional Commits](https://www.conventionalcommits.org/) for all commit messages.
- One logical change per commit, one commit per PR.
